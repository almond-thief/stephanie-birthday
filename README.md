# Stephanie Birthday Surprise

Premium vanilla HTML/CSS/JS birthday page.

## How to use

1. Replace the photo placeholder with Stephanie's photo:
   - Put the image inside `assets/`
   - Rename it to `photo.jpg`
2. Open `index.html` locally to preview.
3. Upload all files to a GitHub repository.
4. Enable GitHub Pages: Settings → Pages → Deploy from branch → `main` → `/root`.

## Files

- `index.html` — page structure
- `style.css` — premium responsive styling and animations
- `script.js` — scene transitions, confetti, falling petals
- `assets/photo.jpg` — your real photo goes here
- `assets/reference.png` — original reference image

No framework, no build step, no backend.
